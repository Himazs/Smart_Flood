# Smart_Flood
Smart Flood is a mobile and web application designed for both the general public and government officials. This project was developed as part of my final year degree.

In Sri Lanka, many people are affected by frequent flood disasters. After researching this issue, I identified the need for a system that can help communities and authorities respond more effectively.

The purpose of this application is to provide timely information, improve communication, and support decision-making during flood situations, ultimately helping to reduce the impact on affected people.
